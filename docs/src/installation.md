# Installation

## Using Cargo

```bash
# Install from crates.io
cargo install idt

# Install from git repository
cargo install --git https://github.com/sh-cho/idt
```

### Build from Source

Clone the repository and build:

```bash
git clone https://github.com/sh-cho/idt.git
cd idt
cargo install --path .
```

## Using Homebrew

```bash
brew install sh-cho/tap/idt

# or
brew tap sh-cho/tap
brew install idt
```
